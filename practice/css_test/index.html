프론트엔드 모의고사(HTML, CSS 영역)

1.
h1 태그는 문서를 대표하는 단어에 사용된다. 가장 높은 중요도를 나타내는 h1 태그는 한 번 밖에 사용할 수 없다. (맞으면 O, 틀리면 X)
2.
텍스트만을 컨텐츠로 가지는 요소는 height를 설정해주지 않으면 자동으로 설정된다. 이때 높이는 font-size로 결정된다.(맞으면 O 틀리면 X)
3.
box-sizing을 content-box로 지정했을 때 해당 박스의 크기는 border, padding, content의 크기를 모두 포함하여 계산된다. (맞으면 O, 틀리면 X)
4.
다음과 같은 html, css 파일이 있을 때 이미지 태그와 div 태그 사이에 생긴 공백을 없애려고 한다. 주어진 css 파일을 보고 vertical-align에 어떤 값이 들어가야 하는지 구하시오. 
<div class="mother">
  <img src="image1.png" alt="sample" class="sample" /> 
  <div class="child"></div> 
</div>
          
.mother {
  width: 280px;
  margin-bottom: 1rem;
}

.child {
  width: 100%;
  height: 50px;
  background-color: cornflowerblue;
}

.sample {
  vertical-align: [???];
}
4.
다음과 같은 html, css 파일이 있을 때 div에 포함된 이미지를 고려해 border-radius를 설정하려고 한다. 주어진 CSS 파일을 보고 overflow에 어떤 값이 들어가야 하는지 구하시오. 
<div class="mother">
  <img src="image1.png" alt="sample" class="sample" /> 
</div>
          
.mother {
  width: 280px;
  margin-bottom: 1rem;
  border-radius: 50%;
  overflow: [????];
}

.sample {
  width: 100%;
}
